這一章的標題用了一個雙關來引題，除了說明了打者擊球的方式改變，也同時說明推動改變並非由位於上級的管理層發起；而是從球員乃至外部非職業棒球圈內的特殊專業人才引領的「飛球革命」。大聯盟是棒球聯盟的最高層級，要進入的難度當然不言而喻，而球員僅能保握短暫的生涯黃金時期。當進入最高殿堂的球員，面臨短暫低潮時，要輕易放棄多年養成的打擊或投球姿勢，做出大幅度的改變是具有高度門檻的。因為人性總是害怕失去，我們害怕失去地位與成功錯身而過，所以害怕改變。


改變的過程是充滿不確定性再加上人性的牽扯，讓我們難以客觀的分析低潮牽扯的變數。但要是能客觀的分析成功或失敗的因子，或許就能延長甚至改造職業生涯。所幸在科技的發展和引領下，一定程度造就了現在所看到一切的運動發展。


在書中提到， J.D. Martinez 說最好的球員對自己擁有不安全感，因為不想要退步的心態使他們不斷努力，這樣才有可能成為最好。而 J.D. Martinez 覺得 Mookie Betts 就擁有這種特質⋯⋯貝茲跟他們不一樣，他總是說：『嘿，兄弟，我想要搞定這個問題，我還沒有解決辦法。』


## 2013 年的 Justin Turner


首先是 Justin Turner 的故事，在發生改變前他是一位能夠守備多個位子的工具人，沒有突出的打擊表現很快就會喪失在大聯盟的位子。以前，打者接收到的觀念都是，專注在嘗試擊出低空的平飛球就好，全壘打只是額外的紅利、意外的驚喜。然而 Marlon Byrd 相信無論練習還是實戰，都應該嘗試打全壘打，他已經受夠了傳統的打擊思維。在大都會時期，Turner和 Byrd 經常一起討論打擊姿勢，他談到打者的跨步、擊球點的重要性。除了要創造更筆直的動線，使能量的轉移更有效率，揮擊的時候應該想著把擊球點提前，在本壘板前方就打到球。而非像傳統打擊所說，把球跟裡面一點再打，這樣能使身體運作更有條理，也能打出拉打方向的高飛球。


在 2013 年的休賽季，Turner 在 Byrd 的引薦下一同到 Ball Yard 與 Doug Latta 進行合作訓練，而一切的革命也從此開始。2014 年 Turner 跟道奇隊簽下一張帶有大聯盟春訓邀請的小聯盟合約，並靠著在春訓的表現，擠上道奇的正式名單，在球季結束靠著 322 個打席繳出 0.340／0.404／0.493的打擊三圍。在全聯盟所有至少累積三百打席的打者中，Turner 高達 158 的加權得分創造值（wRC+，Weighted Runs Created Plus），高居第九名。


道奇看到了 Turner 的好成績，決定再給他一年的機會。2015年，Turner 在439個打席內，繳出一樣優異的成績單，從 2013 到 2015 年，他的滾地球率大減了12.5%，跌幅為全聯盟第四多。2016 年，Turner 爭取到主力先發的位置，揮出 27 支全壘打。那年球季結束，道奇跟他簽下一紙四年總值六千四百萬美金的複數年合約。


影片裡，他雙手的位置比一月時來得高，他的抬腳也是，有幾次揮棒，他感覺都失去平衡，揮完棒時腳踏在想像的打擊區之外。透納抬腳完、揮出球棒的過程中，先把重量導向後腳，再去推動身體往前，這是拉塔最希望選手改掉的習慣。

## 2018 年的 Mookie Betts


章節一開始從 Mookie Betts 在 2018 年的變革說起，這時的 Mookie 擁有極佳的防守能力，但在打擊表現上卻相較去年出現下滑的狀態。2017年年末，紅襪找來 Tim Hyers 擔任總打擊教練，希望能重新建構球團的打擊哲學。這時隊上已經出現因改變打擊動作而獲得重生的 J.D. Martinez ，再加上 Mookie 本身也在尋求改變，在這樣的機運下改變紅襪的命運悄然而生。


球季剛開始，紅襪恰好來到洛杉磯作客。雖然春訓期間已經在 Tim Hyers[^1]的指導下調整了打擊姿勢，但要成為更好只能不斷往正確的方向努力。在和天使比賽的前一天，Mookie 在打擊教練 Tim Hyers 的介紹下來到 Ball Yard，與那位成功改造 Justin Tuner 的「打者溝通師」Doug Latta 會面，直搗問題的核心，調整身體的動作以達到協調和平衡。


結果在會面結束的隔一天 4/17，擔任開路先鋒的 Mookie 與大谷祥平對決的第一個打席就敲出全壘打。那一場更是有著恐怖的表現，在面對不同投手，接連打出三支全壘打。

從影片中可以看到，打者的雙手低於來球，能製造向上的揮棒軌跡。由於投手投球的地方是高於地面的投手丘，加上地心引力的牽引，所以投手投出的球，都是以向下的角度飛行。
如果採水平的揮棒軌跡，那水平的球棒軌跡面跟向下的來球移動路徑，就很少重疊；反之，若採取向上的揮棒軌跡，就能增加球棒擊中來球甜蜜點的機率。
{{< /alert >}}


最終在那一年，Mookie Betts，生涯第一次進入世界大賽就奪下冠軍，並以 .346 打擊率，.640 長打率帶著 129 分的跑分，獲選年度美聯 MVP、金手套以及銀棒獎。這是美聯史上第一人同時獲得三個成就的選手，堪稱生涯最輝煌的一年[^2]。而紅襪在例行賽豪奪 108 勝，刷新了隊史紀錄自 [1912](https://en.wikipedia.org/wiki/1912_Boston_Red_Sox_season) 年創下的 105 勝紀錄。


跟道奇的透納一樣，馬丁尼茲開始在紅襪隊散佈改造新打擊機制的種子。馬丁尼茲很熱衷於分享知識與教學，也在休息區協助海爾斯傳授打擊理念。因為他也是因改變打擊動作而獲得重生的案例，自 2013 年升上大聯盟一直都是替補級表現的球員。在透納尋求突破的同一個冬天，馬丁尼茲也想創造新的打擊機制。在 Craig Wallenbrock 和范史考伊克的協助下，馬丁尼茲在洛杉磯郊區的一處訓練設施，成功打造了他心目中理想的揮棒動作。改造成功的幾年後，他跟紅襪在 2017 到 2018 年的冬天，簽下一紙五年一億一千萬美金的合約。


2017 和 2018 兩年合計，共有260位累積至少150個擊球的打者，在這群打者當中，貝茲、紅襪中外野手布萊德利（Jackie Bradley Jr.）、紅襪游擊手波賈茲（Xander Bogaerts）等三人的強擊球比例增幅（2017至2018年），分別排在第4、第8、第11名。此外， 貝茲、波賈茲、布萊德利的出色擊球（barrel）比例增幅（2017至2018年），也能依序排在全聯盟第1、第2、第13。


## 關於拉塔—一切的開始都始於熱愛棒球


Doug Latta 的初衷並不是要改變棒球，他只是單純熱愛這個運動。據他的說法，他從小生長於困頓的家庭經濟環境，直到高中才第一次加入公立社區學院洛杉磯城市學院（Los Angeles City College）的校隊，之後再到加州路德大學（California Lutheran University）的球隊打球。離開大學的拉塔繼續投身棒球，參加業餘成棒聯盟的隊伍─帕薩迪納紅鳥隊（Pasadena Redbirds）。


雖然拉塔的球員生涯到1990年代尾聲已經告終，但他覺得自己也有一些關於棒球的心得跟資訊可以分享。在洛杉磯，他將新買下建物的一半改造成室內打擊場，並將這個訓練空間命名為棒球園地。他有足夠的實體空間跟別人分享棒球知識，跟好友 Craig Wallenbrock 一樣，拉塔也想開班授課。飛球革命的起始點可以說就是在棒球園地。有許多認同飛球革命理念的打擊教練和支持者都在這裡聚集，討論彼此的想法。


Marlon Byrd 在大都會把在棒球園地學到的知識傳給 Justin Turner 和 Daniel Murphy。之後透納在道奇隊休息室裡佈道，把這些已經改變他人生的理念跟棒球知識，交給天賦異稟的年輕打者 Cody Bellinger 和 Corey Seager，甚至影響到一些打擊教練，如 Tim Hyers。

## 飛球革命


要突破或改變擁有百年運動歷史的聯盟發展體制並不是一件容易的事，有數十年間大聯盟球隊關於球技指導和重大決策的權力位階，都是由退役球員出任的教練享有最高地位。或許正是因為剛好在 2015 年StatCast 大聯盟的數據上線後，實際的資料證實—飛球隊贏球的幫助勝過滾地球。


新打擊姿勢的傳佈變得更快，從數據發現在 -30° ~ 30° 的擊球仰角區間裡（擊球仰角 0° 即為平飛球），擊球仰角每增加十度，全聯盟的加權上壘率就會增加，而且很快就超過全角度的平均值0.315。（全壘打的平均擊球仰角為28.2°。）除了愈來愈多人意識到飛球與優質打擊表現之間的高度關聯性，2015 年季中大聯盟官方用球的組成沒來由地出現變化，導致球在空中容易飛得比較遠，種種因素都使打者有愈來愈多的誘因追求由下而上的揮棒。


2015 年 Statcast 在上線後，從擊球的數據能發現每年的平均擊球仰角都在增加。


| 年份   | 平均擊球仰角 |
| ---- | ------ |
| 2015 | 10.5°  |
| 2016 | 10.8°  |
| 2017 | 11.1°  |
| 2018 | 11.7°  |


同期間，擊球仰角十度以上的擊球比例也上升了 3.3%。根據「棒球指南」提供的資料，2018 年大聯盟的滾地球率寫下自 1950 年有統計資料以來的最低。2017 年，大聯盟打者一共揮出 6105 支全壘打，締造單季新記錄，比 2000年創下的原紀錄還多了 412 支。那一年，以及 2018 和 2016 年，分別是大聯盟史上飛球出牆率（全壘打除以界內飛球的比例）最高的前三名年份，甚至超越了所謂的「禁藥年代」（steroid era）。


在 2017 年，作者刊登在 FanGraphs 的[一篇文章](https://blogs.fangraphs.com/can-more-mlb-hitters-get-off-the-ground/)被道奇棒球事務總裁相中，因而開啟道奇隊內部的話題。 Craig Wallenbrock 時任道奇隊顧問，而他的追隨者 Scott Van Scoyoc 也正式進入球隊管理階層。隨著時代演進，當初在球員之間的口耳相傳的新思維、新觀念，正式進入到球團和農場系統中，而飛球革命不過是那些打破既有產業制度的其中一個新創事業而已。
